# flutter_search_app

Flutter를 기반으로 하는 검색앱입니다

## 설치방법

1. 이 저장소를 클론합니다
   ```bash
   git clone https://github.com/yourusername/flutter_book_search_app2.git
   ```

2. 프로젝트 디렉토리로 이동합니다
   ```bash
   cd flutter_book_search_app2
   ```

3. APP을 실행합니다
   ```bash
   flutter run
   ```

### 사용기술
Flutter, Riverpod
